# Distance.js

A Small Utility to calulate distance from a latitude and longitude points.

```javascript
Distance().from([lat, lng]).to([lat, lng]).unit('miles')
// returns kilometers by default
```

##### Available length units

- kilometers
- miles
- feet

#### more methods to be added
